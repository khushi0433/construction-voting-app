# Project Overview

This repository is a close-to-build starter for a web-based construction partner voting app.

## Users
- 1 admin
- 6 default partners

## Core workflow
1. Admin creates a project
2. Admin adds segments and exactly 3 options per segment
3. Admin assigns partners
4. Partner logs in and scores each option from 1 to 10
5. Partner locks ballot
6. Admin reviews results and audit trail
7. Admin exports proof files
